## üí° Design Philosophy

### What Makes These Templates Enterprise-Grade?

#### 1. **Typography Hierarchy**
- Clear distinction between headers, body text, and data
- Optimal font sizes for readability (12-14px body, 24-36px headers)
- Professional font families (Inter for modern, System fonts for traditional)

#### 2. **Color Psychology**
- **Blue**: Trust, stability, professionalism (Corporate)
- **Gray/Black**: Sophistication, neutrality, elegance (Minimalist, Premium)
- **Purple**: Creativity, luxury, innovation (Creative)
- **Green**: Growth, stability, nature (Traditional Forest)
- **Red/Orange**: Energy, warmth, attention (Creative)
- **Teal**: Modern, fresh, calming (Creative)

#### 3. **White Space**
- Generous padding and margins
- Breathing room around elements
- Never cramped or cluttered

#### 4. **Data Clarity**
- Tables with clear headers
- Aligned numbers (right-aligned for amounts)
- Visible but subtle borders
- Clear total highlighting

#### 5. **Professional Structure**
- Company info ‚Üí Invoice meta ‚Üí Bill To ‚Üí Items ‚Üí Totals ‚Üí Notes
- Logical flow that matches user expectations
- Consistent positioning across all templates

## üöÄ Usage in Code

### Basic Usage

```typescript
import { TemplateRenderer } from '@/components/templates/template-mapper'

const invoiceData = {
  invoiceNumber: 'INV-001',
  date: '2025-01-24',
  // ... other invoice data
}

// Render a template
<TemplateRenderer 
  templateId="minimal-mono" 
  data={invoiceData} 
/>
```

### Check if Template Exists

```typescript
import { templateExists } from '@/components/templates/template-mapper'

if (templateExists('minimal-mono')) {
  // Template is available
}
```

### Get All Templates

```typescript
import { getAllTemplateIds, getTemplateCount } from '@/components/templates/template-mapper'

const allTemplates = getAllTemplateIds()
const count = getTemplateCount() // Returns 25
```

### Get Template Metadata

```typescript
import { TEMPLATE_REGISTRY, getTemplateById } from '@/components/templates/template-registry'

const template = getTemplateById('minimal-mono')
console.log(template.name) // "Minimal Mono"
console.log(template.category) // "minimalist"
console.log(template.suitable) // ["Tech", "SaaS", "Consulting", "Design"]
```
